✦ Questions

FAQ's

What is the Pippin NFT Collection?

The Pippin NFT Collection is a limited edition set of 3333 unique 3D characters. Each character is generated from over 200 hand-crafted traits, making each Pippin entirely one-of-a-kind. This is the founding collection of the Pippin universe.

What blockchain is Pippin on?

Pippin is built on Ethereum. NFTs are ERC-721 tokens stored permanently on-chain. All metadata is stored on IPFS to ensure permanent decentralized storage of your Pippin.
